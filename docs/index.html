GhostRoute

Drop-in script to hide .html extensions on static sites. Clean URLs, no backend required.

Add this in your HTML before closing </body> tag.

No Config Needed

Just drop in the script. No build steps, no server, no settings to mess with.

Static Hosting Friendly

Works on GitHub Pages, Netlify, Vercel, Firebase Hosting, or anywhere else.

SEO and User Friendly

Cleaner URLs are easier to remember, share, and improve your site’s SEO.

Apache Support

Use ghostroute-gen to generate a .htaccess file for extensionless URLs.
